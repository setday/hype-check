# Быстрый старт (Quickstart)

> [!WARNING]
> Документация устарела — проект переведён в шаблонную версию (ml-template).
> См. README.md на английском для актуальной информации.

Проект переведён в формат **переиспользуемого шаблона** для конфигурируемого обучения через Hydra + Lightning.
Все доменно-специфичные компоненты (диффузионные модели, SID-токенизаторы, датасеты Amazon, beam search) заменены на минимальные шаблонные реализации.

Для адаптации под новую задачу смотрите:
- `src/datasets/template_dataset.py` — шаблон датасета
- `src/models/template_model.py` — шаблон модели
- `src/metrics/template_metrics.py` — шаблон метрик
- `config/` — конфигурации компонентов

```bash
# Запуск шаблонного примера
python train.py
```
